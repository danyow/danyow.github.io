4. Python的特征数据类型​

  • 列表和字典的 clear() 方法
a = b = [1, 2, 3]
print(a, b)
a.clear()
print(a, b)

c = d = {'x':1, 'y':2}
print(c, d)
c.clear()
print(c, d)

'''
[1, 2, 3] [1, 2, 3]
[] []
{'x': 1, 'y': 2} {'x': 1, 'y': 2}
{} {}
clear可以理解为是清除指针指向所在地址的内存
'''
