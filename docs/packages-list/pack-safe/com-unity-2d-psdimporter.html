2D PSD Importer

com.unity.2d.psdimporter

描述​

这是用于导入 Adobe Photoshop PSB (Photoshop Big) 文件格式的 ScriptedImporter。此 ScriptedImporter 当前面向想要使用 Unity 2D Animation Package 创建多精灵角色动画的用户。

版本信息​

Compatible for Unity​

Package version 7.0.0-pre.4 is compatible for Unity Editor version 2022.1.

与 Unity 兼容​

These package versions are available in Unity version 2022.1:

文档位置：状态可用版本：
com.unity.2d.psdimporter@7.0兼容7.0.0-pre.4
